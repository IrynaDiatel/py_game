# py_game

2D game developed with Python and Pygame.

## Installation

Clone the repository:

```bash
git clone https://github.com/IrynaDiatel/py_game.git
cd py_game
```

Install dependencies:

```bash
pip install pygame
```

## Run the game

```bash
python main.py
```

## Controls

- ↑ Move up
- ↓ Move down
- ← Move left
- → Move right

## Objective

- Avoid enemies.
- Collect bonuses to increase your score.
- The game ends when the player collides with an enemy.

## Technologies

- Python
- Pygame
